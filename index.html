HTML Basics

The "frontend" of a webpage consists of:

  • HTML = HyperText Markup Language
    • content
  • CSS = Cascading Style Sheet
    • style
  • JS = Javascript
    • interactivity

Volcabulary & Syntax

  • horizontal rule (line):<tag attribute="value">
    • content</tag>

Block vs. Inline

  • block-level elements always start on a new line and take up the full width available
  • inline-elements do not start on a new line and only takes up as much width as necessary

Block-level elements

  • heading:horizontal rule (line):<hl></hl>
    • through<h6></h6>
  • horizontal rule(line):horizontal rule (line):<hr>
  • paragraph:horizontal rule (line):<p></p>
  • ordered list:horizontal rule (line):<ol></ol>
  • unordered list:horizontal rule (line):<ul></ul>
  • list item:horizontal rule (line):<li></li>
  • form:horizontal rule (line):<form></form>
  • div(container):horizontal rule (line):<div></div>

Inline elements

  • line break:horizontal rule (line):<br>
  • image:horizontal rule (line):<img src="URL">
  • link(anchor):horizontal rule (line):<a href="URL"></a>
  • code:horizontal rule (line):<code></code>
  • bold:horizontal rule (line):<strong></strong>
  • italicized:horizontal rule (line):<em></em>
  • span(container):horizontal rule (line):<span></span>
